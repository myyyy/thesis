.. seealso::

    如果你是初学 Python_,你也许会想从了解这门语言开始。Django 是 100% 基于
    Python 的， 因此，如果你对 Python 有坚实的基础，那么你必定将会使用 Django
    得心应手。

    如果你是编程新手，那么你最好先看看 `编程新手 Python 资源列表`_

    如果你有其他语言的编程经验并想快速学习 Python ，那么我们推荐 《Dive Into Python》 ( `Python 2`_ 版本, 
    `Python 3`_ 版本, 还有 `纸质的书籍`_). 如果这本书不符合你的要求，这里还有许多其他 `关于 Python 的书`_.
